Manual Steps & Walkthrough

  1. Elastic Cloud Setup: Sign up for Elastic Cloud (free tier). Create a deployment and note APM Server endpoint, Elasticsearch, and Kibana URLs.
  2. GCP Console: Access Google Cloud Console. Create a GKE cluster (2+ nodes), Cloud SQL (Postgres), and Memorystore (Redis).
  3. Kibana Dashboards: In Kibana, go to Stack Management → Saved Objects, click Import, and upload NDJSON files from dashboards/.
  4. Alerting Rules: In Kibana, go to Observability → Rules, click Import, and upload NDJSON files from infrastructure/alerting-rules/.
  5. Validation: Use Kibana APM and Metrics Explorer to verify data ingestion and dashboards.
  6. For screenshots and trace validation: Follow the assessment instructions and upload screenshots as required.

For any step requiring credentials or manual input, follow the official documentation linked above.
